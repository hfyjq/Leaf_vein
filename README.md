最新版本：5.1

更新公告>>>
一.视觉体验方面：1.新增了颜色区分，列表项目一目了然，重要操作及时发现
                           2.修复了功能框错位的问题
                           3.修复了图标错误的问题
二.功能：1.新增了网站管理，同时默认置顶四家常用驱动程序获取/技术支持的地址，支持自定义添加

三.杂项：1.优化了代码执行速度，修复了部分特殊脚本会偶然添加失败的问题
              2.删除了驱动检测的测试端口，本功能将会在后续版本开放支持
              3.解决了Mcafee偶然报毒的问题
              4.修正了有时候验证窗口会错误的出现在屏幕奇怪的地方
              5.解决了一键删除可能会导致崩溃的问题
              6.解决了首次验证后再次打开程序仍然需要验证的问题
